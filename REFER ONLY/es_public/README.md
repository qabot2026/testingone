# es_public — Internet par dikhti files (Public)

Yeh folder **website visitors** ko milta hai — chatbot, dashboard pages, embed.

**Har client edit:** `es_public/client-based/` (company.config.js, bot-configs, demos)

## Kya hai isme?

| Folder / file | Kaam |
|---------------|------|
| **`client-based/`** | **Per-client UI** — company.config, bot-configs |
| `embed.js` | Client sites par chat embed |
| `widget/` | Chat window |
| `dashboard/` | Admin pages |
| `live-agent/` | Agent inbox |
| `forms/` | Contact, appointment forms |

## Private?

- **GitHub:** Repo private rakho → code copy nahi hoga
- **Website:** Chat chalane ke liye ye files browser ko chahiye — ye normal hai

## URL example

```
https://your-site.com/embed.js
https://your-site.com/company.config.js
```

**Passwords / API keys yahan nahi hain.**
