Plain-text templates only — HTML is generated automatically when emails send. Client emails use Email Notifications (recipients). User emails go to the visitor's $session.params.email — enable or disable each template below. Delivery: Email integration.

How to add a new template (developer)
  1. Add key in es_private/lib/email-template-engine.js — TEMPLATE_KEYS, TEMPLATE_META, default body in defaultTemplates(), and TEMPLATE_TRIGGERS.
  2. Wire send in es_private/lib/email-notifications-mail.js and call it from the right hook (e.g. lead-notifications-runner.js or server.js).
  3. Reload this page — new template cards appear automatically from the API catalog.
